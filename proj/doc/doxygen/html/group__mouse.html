◆ mouse_ih()

void() mouse_ih ( )

Handles mouse interrupts (C implementation)

Reads the status register and the output buffer (OB). Reads one packet byte from the output buffer. If there was some error, the byte read from the OB should be discarded. All communication with other code must be done via global variables, static if possible.
